# 3-v-vrsto
Zelo enostavna, zeankrat še brez GUI, igra 3 v vrsto.

Pravila igre:
  Najprej pobereš kodo iz repozitorija (je open source).
  Poženeš kodo (najprej jo je treba prevesti v terminalu z gcc 3vrsto.c) z:

    Windows: a.exe
    Linux: ./a.out
    MacOS: ./a.out

  
Najprej vpišeš številko stolpca, kamor želiš igrati svojo potezo (torej 1, 2 ali 3).
Nato še število vrstice.

Izpisala se bo igralna plošča s tvojo potezo in potezo računalnika.

Ponavljaj ta korak do konca igre.

Igre ni možno zmagati. Računalnik vedno izvede optimalno potezo.

Have fun!
